Creating new function approximation architectures

The fn-approx package defines the top-level class <fn-approx>. To create new approximation architures, subclass this. Subclasses must implement:
  • reset F resets the parameters of the function approximator to their initial values.
  • evaluate F evaluates the function (using the current parameters) at input x.
  • update X Y STEP-SIZE updates the parameters based on observing that the value at x is y.
  • clone F makes a copy of the approximator by copying the parameters.
